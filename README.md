# sqa-associate-skills-md
Associate SQA Engineer Knowledge Hub

A continuously evolving knowledge repository for Software Quality Assurance, aligned with ISTQB CTFL principles, modern QA engineering practices, and AI-assisted workflows.
Built to document practical skills, testing strategies, automation growth, and long-term professional development as an Associate Software Quality Assurance Engineer.

This repository serves as a structured personal knowledge base for:

Software Quality Assurance concepts and practices
Manual and automation testing workflows
ISTQB CTFL-aligned learning and terminology

AI-assisted QA productivity systems
